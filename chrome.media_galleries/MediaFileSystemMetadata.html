MediaFileSystemMetadata class

name ( string ) The name of the file system.

galleryId ( string ) A unique and persistent id for the media gallery.

deviceId ( optional string ) If the media gallery is on a removable device, a unique id for the device.

isRemovable ( boolean ) True if the media gallery is on a removable device.

isMediaDevice ( boolean ) True if the device the media gallery is on was detected as a media device. i.e. a PTP or MTP device, or a DCIM directory is present.
